# 주간 플래너 (Weekly Planner)

요일별로 할 일을 추가,완료체크,삭제하고 `n월m주차.csv` 파일로 저장해서 **여러 주의 플래너를 따로 관리**할 수 있습니다.

## 주요 기능

- 저장된 플래너 **목록 보기 → 불러오기 / 새로 만들기 / 종료**
- 요일별 **할 일 추가** (요일 순서대로 자동 정렬되어 들어감)
- **완료 체크 / 체크 해제** (`v` 표시)
- **할 일 삭제**
- **크게 보기** — 요일마다 박스로 한눈에 보는 화면
- **CSV 저장 / 불러오기** — `scheduals` 폴더에 주차별 파일로 보관

## 개발 환경

-사용 언어: C
-사용 헤더: `stdio.h`, `string.h`, `windows.h`
-운영체제: Windows (`system("cls")`, `windows.h` 사용)

- 실행은 **`scheduals` 폴더가 있는 위치에서** 해야 파일을 찾고 저장할 수 있습니다.

## 폴더 구조

```
project/
├── main.c
├── README.md
└── scheduals/
    ├── 6월3주차.csv
    ├── 7월1주차.csv
    └── ...
```

- 각 플래너는 `scheduals` 폴더 안에 **`n월m주차.csv`** 형식의 파일 하나로 저장됩니다.
- 월과 주차 정보는 **파일 이름**이 기억하므로, 파일 안에는 할 일 목록만 들어갑니다.
- `scheduals` 폴더가 없으면 저장에 실패합니다. **폴더를 미리 만들어 두세요.**

## 사용 방법

### 1) 시작 화면 — 플래너 목록

```
========== 주간 플래너 ==========
[ 저장된 플래너 ]
  1. 6월3주차
  2. 7월1주차
---------------------------------
번호 = 불러오기  /  0 = 새로 만들기  /  -1 = 종료
선택:
```

- **번호** 입력 → 해당 플래너 불러오기
- **0** 입력 → 새 플래너 만들기 (몇 월·몇 주차 입력, 1~12월 / 1~5주차만 허용)
- **-1** 입력 → 프로그램 종료

### 2) 편집 화면

```
========== 6월 3주차 플래너 ==========
[월요일]
   1. [V] 운동하기
   2. [ ] 장보기
[화요일]
   (없음)
...
=======================================
1.할일추가  2.완료체크  3.삭제  4.크게보기  0.저장 후 목록으로
선택:
```

- 1: 할 일 추가 (요일 1~7, 내용 입력)
- 2: 완료 체크 / 해제 (번호 선택)
- 3: 할 일 삭제 (번호 선택)
- 4: 크게 보기 화면으로 이동
- 0: 저장하고 목록 화면으로 돌아가기

> 할 일 번호는 **요일 순서**대로 1, 2, 3... 으로 매겨집니다.
> 저장은 **`0`(저장 후 목록으로)** 을 눌렀을 때 이루어집니다.

### 3) 크게 보기 (`4`번)

요일마다 박스로 나누어 크게 보여줍니다. `0`을 누르면 편집 화면으로 돌아갑니다.

```
  ┌─────────────────────────────────────────────
  │  [ 월요일 ]
  ├─────────────────────────────────────────────
  │      [V] 운동하기
  │      [ ] 장보기
  │
  └─────────────────────────────────────────────
```

## CSV 저장 형식

예시 — `scheduals/6월3주차.csv`

```
요일,완료,할일
월,0,운동하기
월,0,장보기
수,1,팀 회의 준비
```

- **첫 줄**: 제목 줄 (`요일,완료,할일`) — 불러올 때는 건너뜁니다.
- **그 다음 줄들**: `요일,완료여부(0/1),할일내용` 형식
- 완료여부 `1` = 완료(`V`), `0` = 미완료
